# Tic-Tac-Toe (Python Tkinter)

A simple Tic-Tac-Toe game with a graphical user interface built using Python and Tkinter.

![image](https://github.com/Vishakh01/Tic-Tac-Toe/blob/c77dda61538d93a5bec3f59b6c480a5460b74891/Iterface.png)

## Features
- 3x3 clickable game board
- Turn-based gameplay for X and O
- Win and draw detection
- Reset button
- Popup messages for game over

## Requirements
- Python 3.x

## How to Run
1. Clone this repository.
2. Open the project folder.
3. Run the game using:

```bash
python tic-tac-toe.py
```

## Project Files
- `tic-tac-toe.py` - Main game script
